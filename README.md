# **Hepatitis C Prediction**

Predicting patients with hepatitis C disease using various Machine Learning Models.

Tools: Python, Pandas, Numpy, Seaborn, Scikit-Learn, Matplotlib, Scipy, Feature-Engine, Pickle, Streamlit.
